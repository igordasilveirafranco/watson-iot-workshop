# Executando a Demo:
    
1. Crie uma instância do IoT Platform
2. Crie um novo `Device`
3. Copie as credenciais de conexão no arquivo `sample-device.cfg`
4. Execute o script:

```bash
python ibm-watson-iot-demo.py -c mydevice.cfg 
```
